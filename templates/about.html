JSON 2 HTML

JSON Object will be converted to readable HTML format

This project aims to convert JSON Object to HTML representation. Non-technical audience find it difficult to understand JSON representation. This app aids them in learning and better understanding of deploying JSON.

This project has been developed using open-source python web framework FLASK. The design has been implemented using Twitter-Bootstrap and font-awesome library, built over Bootstrap.
